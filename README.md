# RickLauncher

Лаунчер для **Mount & Blade II: Bannerlord** — запускайте разные версии игры и наборы модов («сборки») из одного окна, с раздельными сейвами, настройками и шейдерами.

**Версия: 1.0.8** · English: **[README.en.md](README.en.md)**

## 📥 Скачать

1. Откройте раздел **[Releases](../../releases/latest)** и скачайте `RickLauncher-vX.Y.Z.zip`.
2. Распакуйте в любую папку (например, на рабочий стол).
3. Запустите `RickLauncher.exe`. Устанавливать .NET не нужно — всё уже внутри.

Дальше лаунчер сам сообщит о новой версии и обновится в один клик. Если GitHub недоступен (например, заблокирован) — обновления и наши моды скачиваются с запасного сервера.

## Что умеет

- 🎮 Находит установленные игры Bannerlord и запускает разные версии и сборки модов из одного окна.
- 🎮 Переносит ваши данные из игры (сейвы, настройки модов, список включённых модов с порядком) — можно сразу продолжить игру.
- 🧩 Управление модами: включить/выключить, порядок загрузки, автопорядок, проверка зависимостей и конфликтов.
- ✨ «Наши моды» — установка наших модов (Crash Doctor, RTS Camera Universal, русский перевод TOR, Lore-Hardcore) в один клик.
- 📦 Установка модов из архива, папки или по ссылке. Готовые сборки модов одним файлом (`.rickpack`), часть идёт в комплекте.
- 🛡 Ваши моды и сейвы не мешают друг другу, файлы игры не портятся.
- 🌍 Русский и английский язык (можно добавить свой — файлы в папке `Lang`). Выбор языка — в настройках.
- ⬆ Автообновление в один клик — с GitHub, а если он недоступен — с запасного сервера.

## Требования

- Windows 10/11 (64-бит)
- Установленная игра Bannerlord
