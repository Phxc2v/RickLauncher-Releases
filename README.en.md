# RickLauncher

A launcher for **Mount & Blade II: Bannerlord** — run several game versions and mod sets ("builds") from one window, with separate saves, settings and shaders.

**Version: 1.0.8** · Русский: **[README.md](README.md)**

## 📥 Download

1. Open **[Releases](../../releases/latest)** and download `RickLauncher-vX.Y.Z.zip`.
2. Unzip it anywhere (e.g. onto your desktop).
3. Run `RickLauncher.exe`. No .NET install needed — everything is inside.

The launcher tells you when a new version is out and updates in one click. If GitHub is unreachable (e.g. blocked), updates and our mods are downloaded from a backup server.

## What it does

- 🎮 Finds your installed Bannerlord games and launches different versions and mod builds from one window.
- 🎮 Brings your data over from the game (saves, mod settings, your enabled-mod list and order) — just continue playing.
- 🧩 Mod management: enable/disable, load order, auto-order, dependency and conflict checks.
- ✨ "Our mods" — install our mods (Crash Doctor, RTS Camera Universal, TOR Russian translation, Lore-Hardcore) in one click.
- 📦 Install mods from an archive, a folder or a URL. Share a ready mod set as one file (`.rickpack`); some ship with the launcher.
- 🛡 Your mods and saves stay separate and never corrupt the game's own files.
- 🌍 Russian and English (add your own — files in the `Lang` folder). Pick the language in Settings.
- ⬆ One-click auto-update — from GitHub, or a backup server when GitHub is blocked.

## Requirements

- Windows 10/11 (64-bit)
- Bannerlord installed
